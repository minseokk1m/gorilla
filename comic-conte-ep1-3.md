# 🦍 GORILLA GAME 만화 콘티 패키지 v3 — GPT Image 인풋용
## (Ep 1·2·3 페이지 단위 한국 웹툰 컴포지션)

> **v3 변경사항 (본편 v3 재편 반영)**
> - 캐릭터 전면 개명: 하나 → **선수용**(분석가·혁신수용자), 용식이 → **나대출**(반면교사·빚투·2배 레버리지·모멘텀), 고 박사 → **고집중 박사**(멘토·전도사). 킹덤 비유 = 영주/기사/소작농
> - Ep 1: **3×3 매트릭스(정중앙 = 고릴라 게임 투자자)** 신규 컷 + 데이트레이딩·모멘텀·밸류와 차별화 명시 + 학력 차별 표현 제거("인서울 4년제" → "평범한 직장인")
> - Ep 1 사례를 **AI 주식**으로 재편(친구 3인 조언 = AI 종목), NVIDIA·CUDA 표준 강조
> - Ep 2: **확산이론의 농업 뿌리**(1943 아이오와 잡종 옥수수, Ryan & Gross → Rogers → Moore → Christensen) 신규 컷
> - Ep 3: 사례를 Segway/C3.ai에서 **전기차(BEV) 캐즘 + 하이브리드·EREV 우회 = 캐즘의 증거**로 전면 교체. 캐즘 = 구조적 매출 정체
> - 면책·저작권 한 줄 고지 추가(인트로 카드 하단)

> **v2에서 유지**
> - 한글 텍스트의 em dash(—) 전부 제거. 콜론(:), 쉼표(,), 줄바꿈으로 대체
> - 말풍선·내레이션 종결 온점(.) 제거. 구어체 톤
> - 카드뉴스 톤이 아닌 한국 웹툰 페이지 컴포지션. 페이지당 panel 2~4개, 비대칭 크기, gutter(흰 여백) 명시
> - 시리즈 빌드업: 표지, 인트로 약속 카드, Ep 1 타이틀, 프롤로그 hook

> **사용 안내**: 이 파일을 GPT 채팅에 업로드 후 "PART 0 Bible 확인했어, 페이지 1부터 시작해줘"라고 말해. GPT가 PART 1 캐릭터 시트 → PART 2~ 페이지 순으로 진행. **각 페이지 생성은 1장씩.**
